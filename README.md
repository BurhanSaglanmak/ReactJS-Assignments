# ReactJS Assignments Project

- Axios API ile JSON datası çekildi. (https://jsonplaceholder.typicode.com/users)

- Proje'de UseState, UseEffect hookları ve Ant Design kullanıldı.

# Özellikler

- Delete işlemi.

- Edit ekranı ve editleme.

- Loading ekranı.
